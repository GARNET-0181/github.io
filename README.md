# ミナモア・ekie フロアクイズ 🏢

広島駅ビル「ミナモア」と「ekie」の荷物仕分け業務のための学習用Webアプリケーションです。

ショップ名から階数を当てるクイズ形式で、楽しく効率的に店舗の場所を覚えることができます。

## 🎯 機能

### ✨ 主要機能

1. **クイズモード** 🎮
   - **minamoaクイズ**: B1F〜6Fの店舗が対象
   - **ekieクイズ**: 1F〜2Fの店舗が対象
   - **全店舗クイズ**: すべての店舗からランダム出題（チャレンジモード）
   - 問題数は10問/20問/30問/50問から選択可能

2. **学習モード** 📚
   - 全店舗の一覧表示
   - 建物別フィルター（minamoa/ekie）
   - ショップ名・カテゴリー・フロアで検索
   - フロア順に並び替え

3. **スコア管理** 🏆
   - 正解数・不正解数・正解率の表示
   - 最高得点の記録と更新
   - リアルタイムスコア表示

4. **フィードバック機能** ✓
   - 即座に正解・不正解を表示
   - 正解フロアとカテゴリーを表示
   - ビジュアルなアニメーション

### 📱 対応デバイス

- **スマートフォン**: iOS / Android対応
- **タブレット**: iPad / Android タブレット対応
- **PC**: デスクトップブラウザ対応
- レスポンシブデザインで画面サイズに自動対応

## 🚀 使い方

### クイズモード

1. トップ画面でモード（minamoa/ekie/全店舗）を選択
2. 問題数を選択（10/20/30/50問）
3. 「開始」ボタンをクリック
4. 表示されたショップ名の階数を選択
5. 即座にフィードバックが表示されます
6. すべての問題に答えると結果画面が表示されます

### 学習モード

1. トップ画面の「学習モード（全店舗一覧）」をクリック
2. 建物でフィルタリング（全て/minamoa/ekie）
3. 検索ボックスでショップを検索
4. すべての店舗をフロア順に確認できます

### キーボードショートカット

- **Escキー**: スタート画面に戻る
- **数字キー (1-7)**: クイズ画面で回答選択

- **2F**: 4店舗（おみやげ）

**合計: 15店舗**

**総登録店舗数: 71店舗**

## 🐛 トラブルシューティング

### スコアが保存されない
- ブラウザのLocalStorageが有効か確認してください
- プライベートブラウジングモードでは保存されません

### 画面が正しく表示されない
- JavaScriptが有効か確認してください
- ブラウザのキャッシュをクリアしてください
- ページを再読み込みしてください

### ボタンが反応しない
- 画面が完全に読み込まれるまでお待ちください
- ブラウザのコンソールでエラーを確認してください
