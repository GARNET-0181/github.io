# ミナモア・ekie フロアクイズアプリ (更新版)

広島駅直結の商業施設「minamoa（ミナモア）」と「ekie（エキエ）」の荷物仕分け業務を効率的に学習できるクイズアプリです。

## 🎯 更新内容 (Version 2.0)

### 1. 2F/3F 階層の完全分離
- **2F**: 北・東・西の3エリアに完全分離
  - 2F北: コスメ・ザッカ (25店舗)
  - 2F東: コスメ・ザッカ (16店舗)
  - 2F西: フード・スイーツ (15店舗)
  
- **3F**: 東・西の2エリアに完全分離
  - 3F東: ライフスタイル・ファッション (9店舗)
  - 3F西: ライフスタイル・ファッション (5店舗)

### 2. フロアマップ位置表示機能
- クイズ回答後、該当フロアの簡易マップを自動表示
- ショップの位置を赤い点でマーキング
- おおよその配置が一目で分かる視覚的支援

### 3. ショップデータの更新
- 2F-6Fの全ショップを公式サイトから精査
- 正確な店舗情報と位置データを追加
- 合計100店舗以上のデータベース

## 📋 機能

### クイズモード
- **minamoaクイズ**: B1F〜6Fの店舗
- **ekieクイズ**: 1F〜2Fの店舗
- **全店舗クイズ**: 全フロアのチャレンジモード
- 問題数: 10/20/30/50問から選択可能

### 学習モード
- 全店舗を階数別に一覧表示
- カテゴリーごとに色分け
- 検索・フィルター機能付き

### スコアシステム
- リアルタイムスコア表示
- 正解率の計算
- 最高得点の記録(localStorage使用)

## 🎮 操作方法

1. **モード選択**: minamoa/ekie/全店舗から選択
2. **問題数設定**: 10〜50問から選択
3. **クイズ開始**: ショップ名を見て階数を選択
4. **結果確認**: フロアマップで位置を確認
5. **学習モード**: 全店舗を一覧で確認・復習
